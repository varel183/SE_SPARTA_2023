## b) Teknologi yang Digunakan
- **Bahasa Pemrograman:** TypeScript
- **Framework/Libraries:** Next.js, Tailwind CSS, Shadcn UI
- **VCS:** GitHub
- **Deploy:** Vercel

## c) Cara Menjalankan Program

### Menjalankan di Localhost
Untuk menjalankan program ini di localhost, ikuti langkah-langkah berikut:

1. Clone repository ini ke local machine Anda.
2. Buka terminal dan navigasikan ke direktori proyek.
3. Jalankan perintah berikut untuk memulai server pengembangan:

   ```bash
   npm run dev
   # atau
   yarn dev
   # atau
   pnpm dev
   # atau
   bun dev
   ```

4. Buka [http://localhost:3000](http://localhost:3000) di browser Anda untuk melihat hasilnya.

### Menjalankan di Server yang Sudah Dideploy
Untuk mengakses portofolio ini yang telah dideploy, kunjungi: [https://portfolio-varel.vercel.app/](https://portfolio-varel.vercel.app/)
